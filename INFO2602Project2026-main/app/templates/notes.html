REST (Representational State Transfer) is an architectural style for building web services. A RESTful API exposes resources via URLs and uses standard HTTP methods.

A REST API is defined by its constraints — statelessness, uniform interface, and resource-based URLs.

HTTP Methods

The four main methods map to CRUD: GET retrieves, POST creates, PUT/PATCH updates, and DELETE removes.
